# English Streak - Landing Page

صفحة هبوط احترافية لبوت تيليجرام English Streak الذي يقدم كلمات إنجليزية يومية.

## الملفات

- `index.html` - الصفحة الرئيسية
- `privacy.html` - صفحة سياسة الخصوصية
- `styles.css` - ملف التنسيقات
- `script.js` - ملف JavaScript للتفاعلات

## المميزات

✅ تصميم retro/sci-fi مع خلفية شبكية ثلاثية الأبعاد  
✅ ألوان: أسود داكن، بيج فاتح، برتقالي قوي  
✅ خطوط pixel/arcade للعناوين  
✅ تصميم responsive يعمل على جميع الأجهزة  
✅ أقسام متكاملة: Hero, Features, Premium, How It Works, Testimonials, FAQ  
✅ صفحة سياسة خصوصية كاملة  

## كيفية الاستخدام

1. افتح `index.html` في المتصفح
2. جميع الملفات يجب أن تكون في نفس المجلد
3. لا حاجة لتثبيت أي مكتبات خارجية (جميعها عبر CDN)

## التخصيص

### تغيير الألوان
عدّل المتغيرات في بداية ملف `styles.css`:
```css
:root {
    --color-dark: #0a0a0a;
    --color-beige: #f5f1e8;
    --color-orange: #ff6b35;
}
```

### تغيير النصوص
جميع النصوص موجودة مباشرة في ملف `index.html` ويمكن تعديلها بسهولة.

### إضافة محتوى
- لإضافة ميزة جديدة: انسخ `.feature-card` في قسم Features
- لإضافة سؤال: انسخ `.faq-item` في قسم FAQ
- لإضافة شهادة: انسخ `.testimonial-card` في قسم Testimonials
